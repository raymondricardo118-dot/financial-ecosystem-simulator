# Hedge Fund System Architecture Simulator

## Overview

This project is a **front-office to back-office financial systems simulator** designed to visually represent how institutional hedge fund and investment bank trading platforms are structured.

It demonstrates the end-to-end lifecycle of financial data across:

- Trade execution (Front Office)
- Market data ingestion
- Risk management
- NAV calculation & reconciliation (Middle Office)
- Settlement & reporting (Back Office)

The system is implemented as a **single-page interactive terminal UI simulation**.

---

## 🧠 System Architecture

The platform simulates a real-world hedge fund architecture:

Front Office → Market Data → Risk Engine → Middle Office → Back Office

Each layer represents a core component in institutional trading systems.

---

## 📊 Modules

### 1. Front Office (Trading Desk)
- Trade entry simulation
- Order routing logic
- Execution workflow representation

### 2. Market Data Layer
- Live price feed simulation
- Order book visualization
- Asset price movement updates

### 3. Risk Engine
- Exposure monitoring
- Value-at-Risk (VaR) simulation
- Limit checks and risk classification

### 4. Middle Office
- NAV calculation simulation
- Trade reconciliation logic
- P&L attribution overview

### 5. Back Office
- Settlement workflow simulation
- Clearing process representation
- Reporting status tracking
